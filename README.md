# war-card-game
War Card Game

## Getting Started
Enter the client directory
```
cd client
```

Install the needed dependencies
```
npm i
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
